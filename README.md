Sudoku-JS
=========

Browser-based sudoku game using RaphaëlJS 2.1.0 and jQuery 1.7.2

http://sudoku.subtledetour.com/

Features:
- Written in SVG, HTML 5, CSS 3 and javascript
- Cross-browser compatible
- Touch support for tablets and smart phones
- Options for different types of players (enable / disable timer, cell highlighting, number placement feeback, penalties)
- Generates random boards using backtrack algorithm
- Saves games to local storage
- Notes mode
